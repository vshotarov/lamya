API Reference
=============
The full reference to pygeon's API.

- `pygeon.content_tree`_ - tree implementation for parsing and managing content
- `pygeon.content_processing`_ - utility functions for extracting excerpts
  and front matter from the content
- `pygeon.config`_ - a base config object to simplify providing arguments to the
  site generation process
- `pygeon.site_generator`_ - the out-of-the-box site generator
